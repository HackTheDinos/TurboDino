# Backyard Expedition App
Install these first:  
NodeJS  
MeteorJS  
Graphicsmagick  


Clone the TurboDino Repo and run the 'meteor' command in the terminal to start the app up.
Follow the steps when trying to login to facebook and google to set up oAuth.
